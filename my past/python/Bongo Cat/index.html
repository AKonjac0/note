Python Bongo Cat

1.键盘监听

如果使用$Pygame$的键盘事件，就只有在鼠标点进窗口时，程序才能接收到键盘输入，如果此时你正在用其他软件，那么程序无法接收到键盘信号。所以，我们要改用后台监听，监听键盘的一举一动。~~(这tm不是病毒吗)~~

因此我们使用$pynput$库。

from pynput import keyboard
from pynput import mouse


def listen_key_block():  
    with keyboard.Listener(
            on_press=on_press, on_release=on_release) as listener:
        listener.join()  # 加入线程池，阻塞写法


def listen_key_nblock():  # 可以和其他函数并行运行
    listener = keyboard.Listener(
        on_press=on_press, on_release=on_release
    )
    listener.start()  # 启动线程

listen_key_nblock()
while True:
    pass

注意第一种阻塞写法的意思是阻塞主线程，直到当前子线程运行完毕。所以它不能与其他程序并行。

而第二种写法可以解决这个问题,就是要加一个无限循环维持程序的执行。

所以，某种程度上来说，阻塞写法也算一个无限循环。

2.窗口创建 & 绘图

使用$Pygame$即可创建任意大小的窗口，并在左上角绘制图片。

# canvas.py

import pygame as pg

class Background():
    def __init__(self, screen):
        self.screen = screen
        self.image = pg.image.load('images/bongo_cat.png')# 初始化定义为抬起手的图片
        self.rect = self.image.get_rect()  # 获取矩形
        self.rect.x = 0  # 固定坐标为左上角
        self.rect.y = 0

    def draw(self, image):
        self.image = pg.image.load(image)  # 加载图片
        self.screen.blit(self.image, self.rect)  # 绘图


# bongo_cat.py
import pygame as pg
import sys
from canvas import Background


def run_game():
    interval = 0.03
    pg.init()
    screen = pg.display.set_mode((300, 177))
    pg.display.set_caption('bongo_cat')
    bgc = (230, 230, 230)
    bg = Background(screen)
    while True:
        for event in pg.event.get():
            if event.type == pg.QUIT:  # 检测右上角鼠标点击退出事件
                sys.exit()
        screen.fill(bgc)  # 背景色
        bg.draw('images/bongo_cat.png')
        pg.display.flip()  # 更新当前图层
        time.sleep(interval)  # 时间间隔，即帧率


run_game()

3.状态切换

当点击键盘时，$pynput$的$listener$会检测到参数$key$，里面包含了你按了哪个键。所以可以用一个列表存储下字符串$keys$,其中$keys = str(key)$。

进一步地，我们用一个字典存储七个按键在一个时间间隔内对应的点击次数，每个间隔结束后清零。

但是有个问题，键盘的输入是，先接受一次$on_press$事件，如果一段时间不松手，才会连续接受$on_press$。这就导致每次按下键盘，小猫的手都会先闪一下，才会持续按下，很不流畅。

所以，我们改成$on_release$出发后再将字典中对应的按键的值清零。这样画面会比较流畅。

而所有图片和这个字典都存储在一个$tmp$文件中，方便查表。

至此，我们的程序就完成了。

from pynput import keyboard

import image_tmp as imt

def on_press(key):
    keys = str(key)
    if keys == 'Key.left':
        imt.tmp['l'] += 1
    elif keys == 'Key.right':
        imt.tmp['r'] += 1
    elif keys == 'Key.down':
        imt.tmp['d'] += 1
    elif keys == 'Key.up':
        imt.tmp['u'] += 1
    elif keys == 'Key.shift':
        imt.tmp['s'] += 1
    elif keys == "'z'":
        imt.tmp['z'] += 1
    elif keys == "'r'":
        imt.tmp['R'] += 1


def on_release(key):
    keys = str(key)
    if keys == 'Key.left':
        imt.tmp['l'] = 0
    elif keys == 'Key.right':
        imt.tmp['r'] = 0
    elif keys == 'Key.down':
        imt.tmp['d'] = 0
    elif keys == 'Key.up':
        imt.tmp['u'] = 0
    elif keys == 'Key.shift':
        imt.tmp['s'] = 0
    elif keys == "'z'":
        imt.tmp['z'] = 0
    elif keys == "'r'":
        imt.tmp['R'] = 0

def listen_mouse_block():
    with mouse.Listener(
            on_move=None, on_click=on_click, on_scroll=on_scroll) as listener:
        listener.join()

4.总结 & 一些坑点

  1. 一开始主循环没有$time.sleep()$,导致$cpu$占用$20\%$

  2. 写$listener.py$时被windows安全中心当成病毒隔离了就nm离谱
